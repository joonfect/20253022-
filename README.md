# SimpleTodo Local

브라우저 LocalStorage 기반의 개인용 할 일 관리 서비스입니다.

## 주요 기능

### 핵심 기능
- ✅ 할 일 추가: 제목, 마감일, 알림 설정
- ✏️ 할 일 수정: 제목, 마감일, 알림 여부 수정
- 🗑️ 할 일 삭제: 삭제 확인 후 즉시 삭제
- ☑️ 완료 처리: 체크박스로 완료/미완료 토글

### 부가 기능
- 🔍 필터링: 전체 / 진행중 / 완료 / 오늘 마감 / 지연됨
- 📊 정렬: 생성일순 / 마감일순 / 이름순
- 🔔 알림: 마감 10분 전 및 마감 시간 브라우저 알림

## 사용 방법

1. `index.html` 파일을 브라우저에서 엽니다.
2. 할 일을 입력하고 마감일과 알림 여부를 선택한 후 "추가" 버튼을 클릭합니다.
3. 필터와 정렬 옵션을 사용해 할 일을 관리합니다.
4. 알림 기능을 사용하려면 브라우저의 알림 권한을 허용해야 합니다.

## 기술 스택

- HTML5
- CSS3 (반응형 디자인, 애니메이션)
- JavaScript (ES6+)
- LocalStorage
- Notification API

## 브라우저 호환성

최신 버전의 다음 브라우저에서 동작합니다:
- Chrome/Edge (권장)
- Firefox
- Safari
- Opera

## 데이터 저장

모든 데이터는 브라우저의 LocalStorage에 저장되며, 다음 형식을 따릅니다:

```json
[
  {
    "id": "1682434830392",
    "title": "예시 할 일",
    "completed": false,
    "createdAt": 1682434830392,
    "dueDate": "2025-11-27T18:00",
    "notificationEnabled": true
  }
]
```

## 알림 기능

- 마감 시간 10분 전 알림
- 마감 시간 알림
- 알림은 브라우저의 Notification API를 사용합니다
- 페이지를 닫아도 알림이 작동합니다 (브라우저가 열려있는 경우)

## 주의사항

- 알림 기능은 브라우저가 열려있을 때만 작동합니다
- LocalStorage 데이터는 브라우저별로 독립적으로 저장됩니다
- 브라우저의 데이터를 삭제하면 저장된 할 일도 모두 삭제됩니다
